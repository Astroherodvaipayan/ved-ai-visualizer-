## ✨ Features

### 🔬 Interactive Periodic Table
- **118 Elements** with detailed information
- **3D Atom Models** with animated electron orbitals
- **Element Search** with instant filtering
- **Color-coded** by element category

### 🧪 Chemistry Tools
- **Chemical Equation Balancer**
- **Molar Mass Calculator**
- **Empirical Formula Finder**
- **Ion Reference Table**
- **Reaction Predictor**

### 🌐 Indian Language Support
- English, हिंदी, ಕನ್ನಡ, தமிழ், తెలుగు

### 🎨 Indian-Inspired Design
- Blue, green, golden, and white color scheme
- Lotus logo representing Indian culture

## 🚀 Getting Started

```bash
git clone https://github.com/Astroherodvaipayan/ved-ai-visualizer-.git
cd ved-ai-visualizer-
npm install
npm run dev
```

Open your browser at `http://localhost:5173`
